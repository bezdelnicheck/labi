1. Создайте публичный репозиторий на GitHub или Bitbucket и добавьте файл 
.gitignore. В этом репозитории будут храниться все лабораторные работы, 
которые будут разрабатываться в дальнейшем. 
2. Авторизуйте открытый ключ на GitHub или Bitbucket. 
3. Клонируйте созданный репозиторий на свой компьютер. 
4. Создайте новую ветку с именем "labs_git" и перейдите на нее. Эта ветка 
будет основной для сдачи задания. 
5. Создайте новую ветку с именем "task1" и перейдите на нее. Эта ветка будет 
основной для выполнения задания. 
6. Находясь на ветке "task1", создайте папку labs. 
7. В папке labs создайте файл с именем lab1.txt и поместите в него половину 
текста задания лабораторной работы. 
8. Сделайте коммит с именем "add first half of labs" и отправьте его на сервер.
